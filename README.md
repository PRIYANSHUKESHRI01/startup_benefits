## Product Overview

StartupBenefits is a full-stack platform enabling early-stage startups to access premium SaaS tools at discounted or free rates. The system emphasizes product clarity, security, scalability, and a premium user experience.

**Target Users:**

- Startup founders
- Early-stage teams
- Indie hackers
- Solo entrepreneurs

---

## Key Features

### Frontend

- Animated landing page with particle and 3D elements
- Global page transitions and micro-interactions
- Deal browsing with search and category filtering
- Locked deal visual indicators
- Deal detail page with sticky claim panel
- User dashboard with real-time claim status
- Skeleton loaders and shimmer animations
- Responsive mobile-first design

### Backend

- JWT-based authentication system
- Secure protected routes
- Atomic deal claim logic
